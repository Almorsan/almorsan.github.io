Condicionales

Los condicionales en JavaScript te permiten tomar decisiones en función de una condición. puedes elegir ejecutar una u otra función o conjunto de código. Esto lo vas a ver mucho mejor con un ejemplo:


El siguiente ejemplo muestra como hacer una sentencia IF que consiste en evaluar una condición, si esa condición se cumple, ejecutará un código, si no, ejecutará otro código distinto

let edad = 18;
if (edad > 17) {
console.log("Eres mayor de edad");
} else {
console.log("Eres menor de edad");

Eres mayor de edad

Si ejecutamos este ejemplo veremos que nos muestra el texto 'Eres mayor de edad' porque ha evaluado la condición que es lo que esta dentro de los paréntesis que dice si edad es mayor que 17, si al comprobarlo resulta que es verdad ejecutará el primer bloque de código, si no, ejecutará el segundo bloque de código que viene justo despues del else, el else indica lo que ocurre cuando no se cumple la condición.


Dentro de los condicionales tenenemos también otra palabra clave que es else if, esta palabra clave lo que nos permite es evular varias condiciones seguidas y que cuando una de ellas se ejecute el resto ya no las ejecuta y de esta forma solo tenemos un else para todas las condiciones que encadenemos.

let edad = 64;
if (edad > 17) {
console.log("Eres mayor de edad");
} else if (edad > 64){
console.log("Perteneces a la tercera edad")
} else {
console.log("Eres menor de edad")

Perteneces a la tercera edad

Aquí vemos que estamos evaluando dos condiciones y que si ninguna de las dos es verdad, ejecuta el bloque de código que viene en el else.
