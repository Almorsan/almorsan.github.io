Las Hojas de estilo en cascada (del ingles Cascading Stylesheets CSS) es la siguiente tecnología que aprenderemos después de HTML. Mientras que HTML se utiliza para definir la estructura y la semántica del contenido, CSS se usa para darle estilo y posicionarlo visualmente. CSS se puede usar, por ejemplo, para cambiar la fuente, el color, el tamaño y el espaciado del contenido, para formar multiples columnas, añadir animaciones y otros elementos decorativos.


Lo primero que debemos aprender para poder usar las hojas de estilo es a como referirnos a un determinado elemento de nuestro HTML, para ello tenemos formas:

Referencia por etiqueta:

Esto es modificar los valores que utiliza por defecto, por ejemplo, si la etiqueta h1 tiene un tamaño de 20px de fuente, podemos indicarle que sea 25px y de color azul celeste y siempre que escribamos una etiqueta h1 tendrá este formato en el HTML que utilice esa hoja de estilo.

h1 {
font-size: 25px;
color: red;
}

Esto es un ejemplo de un texto con la etiqueta h1 modificada.

Referencia por clase

Si queremos referirnos un conjunto de etiquetas o a una etiqueta en concreto que se repite más veces pero solo queremos modificar esa etiqueta en concreto, podemos asignarle una clase y referirnos a esa clase exclusivamente a la hora de aplicar modificaciones, las clases tambien son útiles cuando tienes varios bloques de etiquetas y queremos aplicarles a todo el mismo diseño, podemos ponerles a todos la etiqueta class y el diseño se aplicará a todos ellos.

Para referirnos a una clase ponemos un punto delante del nombre de la clase

.nombreClase {
font-size: 25px;
color: red;
}

Referencia por ID

Por último podemos referirnos a las etiquetas mediantes su ID, la direferencia con la clase es que el valor del ID es único solo podemos poner un mismo ID dentro del HTML, no lo podemos repetir. Para referirnos a un ID dentro del CSS en vez de un punto como en la clase pondremos una almohadilla.

#nombreId {
font-size: 25px;
color: red;
}
