JavaScript es un lenguaje de programación de alto nivel que se utiliza principalmente para agregar interactividad y dinamismo a los sitios web. Es un lenguaje de scripting interpretado que se ejecuta en el lado del cliente (navegador web) y se utiliza para manipular el contenido de una página web, interactuar con el usuario, realizar validaciones de formularios, realizar solicitudes a servidores y mucho más.


Para empezar a hablar de un lenguaje programación, lo primero que debemos conocer son las variables, las variables son como contenedores donde nosotros vamos a meter datos, información. por ejemplo, yo puedo tener una variable llamada numero y meterle en número 25.


Para declarar una variable usaremos las palabras clave 'let' o 'const' (En versiones antiguas también se utilizaba 'var' para declarar variables, pero ya está en desuso) y usaremos let en la mayoría de casos, solo usaremos const cuando quieras que el valor de esa variable nunca cambie.


Dependiendo del tipo de dato que metas dentro de una variable, esta será de un tipo u otro, dentro de los lenguajes de programación tenemos varios tipos de datos como son, Integer(numeros), String(cadenas de texto), Boolean(verdadero o falso) entre otros. Estos son los mas sencillos. A continuación te voy a mostrar unos ejemplos de cada uno.

let numero = 25;
let texto = "Esto es un texto y debe ir entre comillas";
let booleano= true;
