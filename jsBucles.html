Bucles

Los bucles son una estructura fundamental en cualquier lenguaje de programación, estos te permite ejecutar un código repetidas veces mientras se cumpla una condición especiífica lo que nos permite automatizar tareas repetitivas. Dentro de los bucles vamos a hablar de 3:

  • 1. BUCLE FOR



El bucle FOR es una estructura que tiene una variable contador, esta variable contador simplemenete se encarga de contar las vueltas que damos en el bucle, y cuando llega a la cantidad de vueltas que le hemos indicado como límite este finaliza

for (let i = 0; i < 5; i++) {
console.log(i);
}

  • 2. BUCLE WHILE



El bucle WHILE hace lo mismo que el bucle FOR ejecutar el código repetidas veces, mientras se cumpla una condición, la principal diferencia con el FOR es que la condición no tiene por que ser numérica y la debemos indicar nosotros, puede ser si un texto está vacío, si una variable booleana es true o false, lo que nosotros queramos y mientras se cumpla la condición el bucle continua.

let i = 0;
while (i < 5) {
console.log(i);
i++;
}

  • 3. BUCLE DO WHILE



El bucle DO WHILE es igual que el WHILE pero en el bucle WHILE existe la posibilidad de que el bucle no de ni una sola vuelta, dado que hay una condición que si no se cumple, este nunca se ejecuta, esto con el bucle DO WHILe no pasa, tenemos la garantía de que se ejecuta al menos una vez porque la condición que decide si seguimos en el bucle evalua al final del mismo, con lo que el código se ejecuta al menos una vez.

let i = 10;
do {
console.log(i);
} while (i < 5);

10

Solo se muestra el 10, porque como no se cumple la condición ejecuta 1 vez el código dentro del do while y se sale.
