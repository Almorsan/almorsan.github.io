Procesamiento de texto



En este apartado te voy a mostrar como puedes mostrar un texto en una página web con HTML, y modificarlo según el uso que le quieras dar.




La siguiente etiqueta es la más simple para mostrar texto dentro de una página web, es recomendable usar siempre al menos una etiqueta para mostrar un texto, dado que luego podremos hacer modificaciones sobre todos los textos que tengan esta misma etiqueta, como cambiar el color, tamaño, fuente, etc.



<p> Esto es un texto de prueba </p>

El contenido de la etiqueta <p> se muestra exactamente igual que la escribes



Los elementos de encabezado implementan seis niveles de encabezado del documento, h1 es el más importante, y h6, el menos importante. Un elemento de encabezado describe brevemente el tema de la sección que presenta. La información de encabezado puede ser usada por los agentes usuarios, por ejemplo, para construir una tabla de contenidos para un documento automáticamente.

Para poder resaltar un texto mediante el uso de negrita (bold), utilizaremos la etiqueta strong. Para el uso de cursiva(italic), por su parte, utilizaremos la etiqueta em.
